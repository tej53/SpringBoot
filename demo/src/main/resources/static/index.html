Registrar's Office — Records Drawer 4

Student Catalog

One card per student. File, find, amend, or withdraw a record.

POST /create
New Student Card

File a new record into the drawer.

GET /get-students/:id
Find a Card

Pull a record by student ID.

PUT /update/:id
Amend a Card

Revise the details already on file. All fields are saved as entered — fill in every one, including values that stay the same.

DELETE /delete/:id
Withdraw a Card

Pull a record from the drawer for good. This cannot be undone.

Talks to your API over fetch — nothing is stored in the browser.
